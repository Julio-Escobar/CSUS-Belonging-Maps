# CSUS Belonging Maps App
![Belonging Maps Placeholder Logo](/mobile/belonging_maps_app/assets/logo.png)
## About the Project
The CSUS Belonging Maps App is a full-stack Flutter app leveraging the ArcGIS Maps SDK. The purpose of the app is to innovate on the existing Belonging Maps web implementation, creating a more user-friendly and intuitive mobile app. This is intended to broaden the user base and help to build a stronger, more interconnected community of Sac State students in the region. 

## About the Belonging Maps
The Belonging Maps is a collaborative project between Sacramento State's Inclusive Excellence and the Department of Geography. This innovative, multimedia resource helps students, faculty, staff, and prospective Hornets discover places and programs that foster connection, support, and a sense of community. Each community map includes two complementary features: the Campus Belonging Map, which highlights campus spaces, organizations, and services designed to help people thrive at Sac State; and the Regional Community Belonging Map, which introduces users to local businesses, cultural centers, and community organizations that reflect the vibrancy of the Sacramento region. Together, these maps strengthen pathways between the university and the community, supporting student engagement, well-being, and success.
For more information about the Belonging Maps, please visit their site [here](https://www.csus.edu/president/inclusive-excellence/belonging-education-support/belonging-maps.html).

### Welcome Page
The Welcome Page is the first thing you see when opening the app. It gives you the ability to navigate to the Campus and Community maps with the onscreen buttons or navigate to other pages via the hamburger menu in the top left.

### Hamburger Menu
The Hamburger Menu in the top left lets you conveniently navigate to any page in the app.

### Community Maps Directory
The Community Maps Directory gives you the option to navigate to any of our three community maps.
![Community Maps Directory Screenshot](/readme-images/community-maps.jpg)

### Campus Maps Directory
The Campus Maps Directory gives you the option to navigate to any of our three campus maps.
![Campus Maps Directory Screenshot](/readme-images/campus-maps.jpg)

### Map Pages
Once you navigate to a respective community or campus map, you are able to see location pins on an ArcGIS powered map. These pins show the location of local businesses, medical or legal centers, academic resources, and much more. The locations you'll see are loaded dynamically based on on which map you've chosen.
![Map screenshot](/readme-images/maps.jpg)

### Location Info Popup
If you'd like to know more about a location, you can tap on its pin to open a popup window. In this window you'll find helpful information such as the location's name, address, contact info, and more.
![Location Info Popup Screenshot](/readme-images/location-info.jpg)

### Community Resources
The community resources page helps you find resources in the Sacramento community.
